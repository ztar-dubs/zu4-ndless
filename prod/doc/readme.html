Ultima IV - TI-Nspire CX CAS Port

A port of Ultima IV: Quest of the Avatar to the TI-Nspire CX CAS calculator, based on the zu4 (Zesty Ultima IV) open source engine.

Requirements

  • TI-Nspire CX or CX CAS (color screen required)
  • Ndless jailbreak installed on the calculator
  • Ultima IV PC game data (included as ultima4.zip.tns)
  • VGA upgrade graphics (included as u4upgrad.zip.tns)

Installation

Copy the entire ultima4 folder to your calculator's /documents/ directory, preserving the directory structure below:

documents/ ultima4/ ultima4.tns <-- main executable conf/ <-- configuration files (XML) config.xml.tns creatures.xml.tns graphics.xml.tns maps.xml.tns tileRules.xml.tns tileset-base.xml.tns tileset-dungeon.xml.tns tilesets.xml.tns tilemap-base.xml.tns tilemap-dungeon.xml.tns vendorScript.xml.tns music.xml.tns sound.xml.tns dtd/ <-- DTD files *.dtd.tns data/ <-- game data archives ultima4.zip.tns u4upgrad.zip.tns graphics/ <-- tile graphics (PNG) ega/ *.png.tns vga/ *.png.tns u4.png.tns
Important: All files must have the .tns extension. The directory structure must be preserved exactly as shown.

To launch, open ultima4.tns from the calculator's file browser.

Special Keys (Shift = Alt)

The calculator has no Alt key. Use Shift instead:

KeyAction
Shift + HHelp (key reference)
Shift + QQuit to Main Menu
Shift + VShow Version
Shift + XQuit Game
Note: Sound and music are not available on the TI-Nspire. The volume-related keys (v, comma, period, <, >) are disabled.

Save Files

The game creates the following save files in the ultima4/ directory:

  • party.sav.tns — Party data and game state
  • monsters.sav.tns — Monster positions
  • outmonst.sav.tns — Overworld monster positions
  • dngmap.sav.tns — Dungeon map data

Troubleshooting

  • "Memory allocation failed" — Reboot the calculator and try again. RAM may be fragmented after multiple app launches.
  • Missing images / blank screen — Verify that all files are present with their .tns extension and the directory structure is correct.
  • Game won't start — Make sure Ndless is properly installed and the calculator has a color screen (CX or CX CAS).
